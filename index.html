🔑 Jak získat API klíč
💡 Jeden vygenerovaný trip stojí cca 5–10 Kč. Za $5 (115 Kč) máš desítky tripů.
1
Vytvoř účet na Anthropic
Přejdi na Anthropic Console a zaregistruj se.
Otevřít Anthropic Console →
2
Dobij kredit
Klikni na Billing → Add credits. Minimum $5 (cca 115 Kč).
Otevřít Billing →
3
Vytvoř API klíč
Klikni na API keys → Create Key. Klíč začíná sk-ant-...
Otevřít API Keys →
4
Vlož klíč do Expedice
Zkopíruj klíč (sk-ant-...) a vlož ho do pole API klíč v nastavení ⚙️.
